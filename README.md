# miRSM R package

# Introduction
This package provides several functions to study miRNA sponge modules.

# Installation
```{r echo=FALSE, results='hide', message=FALSE}
source("https://bioconductor.org/biocLite.R")
biocLite("miRSM")
```

# Usage
```{r echo=FALSE, results='hide', message=FALSE}
library(miRSM)
```
